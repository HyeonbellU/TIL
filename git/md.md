# 오늘의 공부
- git 복습
- git bash 명령어 (CLI명령어)

# 학습 내용
1. l
- \$ : interaction의 준비
- \~ : user folder, 로그인 유저에게 특별한 권한 없이 모든 행동 가능하다.
- \/ : 하드디스크의 최상위 폴더

2. 메인 명령어와 옵션 명령어(flag)
- ls : list, 모든 리스트를 보여준다.
>  -a : 숨겨진 파일까지 보여주기
>  -l : 파일의 세부정보, 권한 설정 확인
>  -al : all files line by line
- pwd : print working directory
- cd *목적지* :change directory
>   \./ : 현재위치
>   .. : 상위 디렉토리
>   
- mkdir : make directory
- touch *파일명.확장자* : 파일 만들기
 + 특정 프로그램을 통해서 실행되는 파일 생성 시 오류가 발생할 수 있기 때문에 텍스트 기반 구동 파일만 생성하도록 하자.
- mv *파일명 목적지* : 
- \* : 와일드카드, 이 부분은 어떤 값이든 올 수 있다.
>  .py로 끝나는 모든 파일 옮기기 : mv \*.py bin
>  m으로 시작하는 모든 파일 옮기기 : mv m\*.\* bin
